# Data Science on Marketing ML

A collection of machine learning projects applied to real-world marketing problems, built as part of my data science studies.

## About

This repository explores how data science and machine learning can be used to solve common marketing challenges — from understanding customer behaviour to predicting churn and analysing campaign performance.

## Projects

| Project | Description | Techniques |
|---|---|---|
| Customer Segmentation | Group customers by behaviour and value | K-Means, RFM Analysis |
| Churn Prediction | Predict which customers are likely to leave | Logistic Regression, Random Forest |
| Campaign Analysis | Analyse which marketing campaigns perform best | EDA, A/B Testing |

## Dataset

This project uses the [Online Retail Dataset](https://www.kaggle.com/datasets/vijayuv/onlineretail) from Kaggle. To run the notebook locally:

1. Download the dataset from Kaggle (free account required)
2. Extract the ZIP file
3. Place `OnlineRetail.csv` inside the `data/` folder

## Tech Stack

- **Language:** Python
- **Libraries:** pandas, numpy, scikit-learn, matplotlib, seaborn
- **Tools:** Jupyter Notebook, GitHub
